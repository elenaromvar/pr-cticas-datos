Uso de la API del Covid19 con Pandas¶

Esta práctica está realizada con Python 3, un lenguaje de programación cuya importancia radica en que la sintaxis dé como resultado un código legible y de fácil comprensión. Python 3 se define según esta fuente. como: "un lenguaje de programación multiparadigma, ya que soporta orientación a objetos, programación imperativa y, en menor medida, programación funcional".
